### iSpace
Computer Programming 2 - Final Project

# Project Setup & Development
1. **Download** all required project files.
2. **Consolidate** all downloaded files into a single, dedicated project folder.
3. **Open** the *iSpace.cbp* project file using **Code::Blocks**.
4. **Begin** coding your changes.

# Uploading Changes to GitHub
1. **Navigate** to iSpace GitHub repository and click on the **Upload files** option.
2. **Upload** the newly updated *main.cpp* file.
3. **Commit** the changes directly to the main branch.
